# Boat Tags (VSC Fork)

Fork of [boattags by xXx_Ape_xXx](https://mods.vintagestory.at/boattags) for the VSC server.

Craft a tag medallion and right-click it onto your sailboat to claim it and track it on the map.

## Changes from upstream

- Non-owners cannot deconstruct a tagged boat. Untagged boats are unaffected.

## Building

```
VINTAGE_STORY="/path/to/Vintagestory" dotnet build
```
